## ðŸŒ Deploying to GitHub Pages

This project is designed to run directly without a build step, making it extremely easy to host on GitHub Pages.

1.  **Push this code** to a GitHub repository.
2.  Go to your repository **Settings**.
3.  Navigate to the **Pages** section (on the left sidebar).
4.  Under **Build and deployment** > **Source**, select **Deploy from a branch**.
5.  Select **main** (or master) branch and **/(root)** folder.
6.  Click **Save**.

Wait a minute, and your game will be live at `https://your-username.github.io/repo-name/`!
